## Banking website using JS
This is a practice project to improve js. This website is simple and design is not all focused here.

### Built with
- HTML5
- Telwind css
- Javascript 


### Features

Users will able to:

- Login and enter the banking page
- Deposit, Withdraw money interatively
- logout
